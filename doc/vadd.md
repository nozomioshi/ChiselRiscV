There are 8 different sizes of adder, from 8-bit to 1024-bit.

```scala
for(vSew <- 0 to 7) {
    val sew = 1 << (vSew + 3)
    val num = Vlen*8 / sew
    when(csrSew === sew.U) {
        vAddvv := Cat(Seq.tabulate(num)(n =>
            (vs1Data(sew*(n+1)-1, sew*n) + vs2Data(sew*(n+1)-1, sew*n))
        ).reverse)
    }
}
```
